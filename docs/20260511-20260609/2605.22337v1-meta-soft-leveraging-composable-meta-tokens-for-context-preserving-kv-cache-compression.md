## 摘要
大型语言模型使用的KV缓存具有线性增长的时间复杂度，因此在处理长上下文时，LLM会面临内存膨胀和解码效率降低的问题。当前的KV缓存驱逐已成为重要的研究方向；然而，基于固定软令牌（如Judge Q）的现有方法依赖静态参数集作为查询来评估KV对的重要性，因此它们无法动态适应不同的输入提示，也无法精确捕捉复杂且变化的任务相关性。此外，被驱逐的KV对会被永久丢弃，导致不可逆的信息丢失和上下文断裂。为解决这一问题，我们提出Meta-Soft，一种基于探测驱动的上下文集成的动态压缩框架。具体而言，我们构建一个包含可学习正交基矩阵$\mathcal{L}$的元库，并使用带有Gumbel-Softmax的选择器网络生成可微的稀疏组合权重，从而从输入提示特征中动态合成最具针对性的$k$个软令牌。我们将这些软令牌附加到输入序列末尾以探测关键信息。我们还引入一种基于注意力流的整合机制，将移除令牌的语义信息重新分配到保留令牌中，从而有效保持被丢弃的上下文信息。在多个数据集上的实验表明，我们的方法优于现有的最先进驱逐方法，并为KV缓存压缩提供了一种新的解决方案。

## Abstract
The KV cache used in large language models has linearly growing time complexity, so LLMs face memory blow-up and reduced decoding efficiency when they process long contexts.Current KV Cache eviction has become an important research direction; however, existing methods based on fixed Soft Tokens (e.g., Judge Q) rely on a static parameter set as the query to evaluate the importance of KV pairs, so they cannot adapt dynamically to different input prompts, and they cannot precisely capture complex and changing task relevance.Also, evicted KV pairs are discarded permanently, so this causes irreversible information loss and context breaks. To address this problem, we propose Meta-Soft, a dynamic compression framework based on probe-driven context integration. Specifically, we build a meta-library with a learnable orthogonal basis matrix $\mathcal{L}$, and we use a selector network with Gumbel-Softmax to produce differentiable sparse combination weights, so we dynamically synthesize the most targeted $k$ Soft Tokens from the input prompt features.We append these Soft Tokens to the end of the input sequence to probe key information. We also introduce an attention-flow based integration mechanism, which redistributes the semantic information of removed tokens into retained tokens, and this keeps the dropped context information effectively.Experiments on multiple datasets show that our method outperforms existing state-of-the-art eviction methods and provides a new solution for KV Cache compression.

- **问题根源**：大语言模型（LLM）推理时使用KV缓存机制避免重复计算，但随着输入序列（长上下文）增长，KV缓存占用的GPU内存线性增长，导致内存溢出和解码吞吐量下降。
- **现有方法局限**：
  - **静态查询**：基于固定 Soft Token 的淘汰方法（如 Judge Q）使用静态参数集评估KV对重要性，无法动态适应不同输入提示，难以捕捉复杂任务相关性。
  - **硬淘汰导致信息丢失**：被判定为“不重要”的KV对直接被永久丢弃，造成不可逆的上下文断裂和幻觉风险。虽有一些KV合并方法（如CaM、ZipCache）试图通过平均或聚类减少KV对，但常导致语义混淆。
- **研究目标**：设计一种动态、上下文保持的KV缓存压缩框架，既能自适应评估重要性，又能在压缩时保留丢弃令牌的语义信息。

### 2. 方法论：核心思想、关键技术细节
- **整体框架**：Meta-Soft 包括离线训练和在线推理两个阶段。离线阶段优化元库（Meta-Library）和选择器网络；在线阶段动态生成输入相关的软令牌（Soft Tokens）进行探测，并根据重要性分区，通过注意力流整合机制将丢弃令牌的信息注入保留令牌。
- **核心技术组件**：
  - **元库与动态合成**：
    - 构建大小为 $M \times d$ 的可学习正交基矩阵 $\mathcal{L}$（$M=512$），存储通用语义原子。
    - 使用选择器网络 $f_\theta$ 结合 Gumbel-Softmax 从输入提示特征中产生稀疏组合权重，动态合成 $k=32$ 个软令牌。训练损失包含均方误差（MSE）和正交正则项，确保探测准确性和特征多样性。
  - **两级训练策略**：
    - **阶段一**：联合优化 $\mathcal{L}$ 和 $f_\theta$（5 epoch），基矩阵学习通用注意力模式。
    - **阶段二**：冻结 $\mathcal{L}$ 和 LLM 骨干，仅微调 $f_\theta$（3 epoch），精化任务特定组合。
  - **软令牌探测与缓存分区**：
    - 将合成的软令牌附加到输入嵌入序列末尾，通过一次前向注意计算获得探测分数 $S \in \mathbb{R}^{k \times L}$，平均得到重要性分布 $A_{\text{soft}}$。
    - 基于 $A_{\text{soft}}$ 和预算 $B$ 进行 Top-K 分区，得到保留集 $I_{\text{keep}}$ 和丢弃集 $I_{\text{drop}}$。
  - **上下文整合（Contextual Consolidation）**：
    - **负载均衡的注意力流聚合**：计算丢弃令牌与保留令牌的键空间相似度 $S_{\text{sim}}$，对每个丢弃令牌选择 top-$m$ 个相似保留令牌，稀疏分配注意力权重。
    - 引入列重加权因子避免少量保留令牌吸收过多质量，然后重新归一化得到流权重 $W_{\text{flow}}$。
    - 聚合丢弃令牌的值 $V_{\text{drop}}$ 到保留令牌：$\Delta V = W_{\text{flow}}^\top V_{\text{drop}}$，并通过负载自适应门控更新保留值：$\hat{V}_{\text{keep},j} = V_{\text{keep},j} + \gamma (g_j \Delta V_j)$，保留键不变以保持位置信息。
- **推理流程**：整个过程仅在预填充阶段完成，软令牌和丢弃集在构建压缩缓存后立即释放，解码阶段无额外开销。

### 6. 主要结论与发现
- **性能优越**：Meta-Soft 在语言建模（PPL最低）、LongBench（平均分最高，保持全KV的92.9%-97.2%）、RULER（所有长度下最佳，135k上下文下仍显著优于其他方法）上均优于所有对比基线。

- **效率可接受**：软令牌生成开销极小（<0.3%预填充时间），端到端延迟与现有压缩方法相当，解码吞吐量提升2.16x，支持更大批大小。
- **无需额外LLM训练**：元库和选择器轻量，训练一次后即插即用。

### 7. 优点
- **动态适应性**：利用输入特征动态合成软令牌，克服了静态查询方法的跨任务失配问题。
- **上下文保持**：通过注意力流整合而非硬丢弃，有效保留了被淘汰令牌的语义信息，减少幻觉风险。
- **设计精巧**：正交基元库提供多样性；负载均衡稀疏路由防止信息覆盖；两阶段训练分离通用和特定知识。
