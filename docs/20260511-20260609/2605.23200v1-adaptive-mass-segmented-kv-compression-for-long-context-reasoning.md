## 摘要
键值（KV）缓存的线性增长是长文本大语言模型推理的关键瓶颈。现有的KV压缩方法通过基于重要性分数驱逐token来缓解这一问题。然而，我们表明它们依赖于全局Top-k选择会触发区域擦除：连续推理块的严重驱逐会破坏逻辑连贯性。为了解决这个问题，我们提出了自适应质量分段（AMS）KV压缩，这是一个将范式从token级竞争转变为区域感知配额分配的框架。AMS根据注意力质量的空间分布自适应地划分KV缓存，确保结构上重要的推理段获得有保证的内存配额。为了确保迭代解码过程中的稳定性，引入了一种基于EMA的平滑机制，以防止段边界抖动。关键的是，AMS是一个通用的即插即用层，与现有的评分器正交。它可以无缝集成到代表性方法中，如TOVA、Expected Attention、KeyDiff、R-KV和TriAttention。AMS还与vLLM等现代分页KV服务框架系统兼容，支持高效的收集和紧凑KV执行，而无需引入额外的稳态注意力开销。在多种任务（包括数学推理（MATH500、AIME、GSM8K）、代码补全、开放域问答和稀疏检索）上进行的大量实验表明，AMS始终能缓解结构碎片化并提升模型性能。

## Abstract
The linear growth of the Key-Value (KV) cache is a critical bottleneck in long-form LLM inference. Existing KV compression methods mitigate this by evicting tokens based on importance scores. However, we show that their reliance on global Top-k selection triggers Region Wipe-out: the severe eviction of contiguous reasoning blocks that derails logical coherence. To address this, we propose Adaptive Mass-Segmented (AMS) KV Compression, a framework that shifts the paradigm from token-level competition to region-aware quota allocation. AMS adaptively partitions the KV cache based on the spatial distribution of attention mass, ensuring structurally vital reasoning segments receive guaranteed memory quotas. To ensure stability during iterative decoding, an EMA-based smoothing mechanism is incorporated to prevent jitter in segment boundaries. Crucially, AMS is a universal plug-and-play layer that is orthogonal to existing scorers. It can be seamlessly integrated into representative methods such as TOVA, Expected Attention, KeyDiff, R-KV and TriAttention. AMS is also system-compatible with modern paged-KV serving frameworks such as vLLM, supporting efficient gather-and-compact KV execution without introducing additional steady-state attention overhead. Extensive experiments across a diverse suite of tasks, including mathematical reasoning (MATH500, AIME, GSM8K), code completion, open-domain QA, and sparse retrieval, demonstrate that AMS consistently mitigates structural fragmentation and boosts model performance.

# 论文详细总结：Adaptive Mass-Segmented KV Compression for Long-Context Reasoning

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：长文本推理（如 Chain-of-Thought 生成）中，KV 缓存线性增长成为主要瓶颈，导致推理延迟高、内存占用大。
- **现有方法局限**：当前解码时 KV 压缩方法（如 TOVA、H2O 等）依赖全局 Top-k 选择，即基于重要性分数逐 token 竞争。这种 token 级竞争会触发 **Region Wipe-out**：连续推理块（如中间推导步骤）被整体驱逐，破坏推理逻辑连贯性，引发问题漂移、过早推翻、重复崩溃等故障模式。
- **研究意义**：亟需一种能保留连续推理区域结构完整性的压缩框架，在有限内存下维持长文本推理的准确性与稳定性。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
将 KV 压缩范式从“token 级竞争”转变为“区域感知配额分配”（allocate-then-score）。先根据注意力质量的空间分布自适应划分序列为区域，为每个区域分配保证保留配额，再在每个区域内基于原有评分器进行局部 Top-k 选择。

### 关键技术细节
1. **头部质量分布计算**  
   - 利用最近 W 步解码查询的注意力分数，经 ReLU + ε 归一化得到非负质量分布 m（和为1）。
   - 质量高的区域（密集推理步）将获得更精细的分段。

2. **自适应分段（Adaptive Segmentation）**  
   - 基于累积质量曲线，按固定质量阈值 Δ 切割序列得到初始段。
   - 应用拆分/合并启发式规则，确保每段长度在 [Lmin, Lmax] 内，高质量区域被切得更细，低质量区域被合并。

3. **段级配额分配（Segment-wise Quota Allocation）**  
   - 每个段首先获得最小保证配额 qmin（保证不被完全擦除）。
   - 剩余预算按各段质量比例分配，并作边界调整以满足总预算 Tkeep。
   - 必须保留起始 sink token 和最近若干 token（must-keep 集）。

4. **段内选择（In-Segment Selection）**  
   - 使用任意现有评分器（如 TOVA、Expected Attention、KeyDiff、R-KV、TriAttention）计算每个 token 的重要性分数。
   - 在每个段内，选择配额数量的最高分 token。最终保留集 = must-keep ∪ 各段选择。

5. **EMA 信用机制（EMA Credit for Temporal Stability）**  
   - 维护一个跨压缩事件累积的历史信用向量 c，更新规则：c ← λc + (1-λ) m^(e)。
   - 使用混合系数 β 融合当前质量与历史信用，得到平滑后的质量分布用于分段和配额分配，减少压缩事件间的边界抖动。

6. **算法流程**（单头、单次压缩事件）：
   - 计算质量分布 → 应用 EMA 信用（可选）→ 自适应分段 → 段级配额分配 → 结合 must-keep 集 → 段内 Top-k 选择 → 收集压缩 KV。
