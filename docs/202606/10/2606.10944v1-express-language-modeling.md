## 摘要
我们引入了一种新工具Express，用于将非因果注意力近似转换为具有匹配近似保证的因果注意力近似。当与最先进的Thinformer近似结合时，Express改进了已知的最佳因果注意力保证，对于长度为n的序列，以仅$O(s)$内存和$O(s^2 \log^2(n))$压缩开销，实现了$\log^{3/2}(n)/s$的近似误差。我们将这些进展与高效的I/O感知Triton实现相结合，展示了相对于FlashAttention 2的显著加速，并利用Express克服了语言建模流程中的四个资源瓶颈：长上下文预填充、KV缓存压缩、长形式内存受限解码和长形式计算受限解码。

## Abstract
We introduce a new tool, Express, for converting a non-causal attention approximation into a causal approximation with matching approximation guarantees. When combined with the state-of-the-art Thinformer approximation, Express improves upon the best known causal attention guarantees, delivering $\log^{3/2}(n)/s$ approximation error with only $O(s)$ memory and $O(s^2 \log^2(n))$ compression overhead for a sequence of length $n$. We pair these developments with an efficient I/O-aware Triton implementation, demonstrate substantial speedups over FlashAttention 2, and use Express to overcome four resource bottlenecks in the language modeling pipeline: long-context prefill, KV cache compression, long-form memory-constrained decoding, and long-form compute-constrained decoding.

- **研究动机**：语言模型中的因果注意力（causal attention）受因果掩码限制，每个查询只关注其之前的键值对，导致计算复杂度与序列长度成二次方关系，成为长上下文预填充、KV缓存压缩、长形式解码等场景的资源瓶颈。现有高效的注意力近似方法（如Thinformer）针对非因果（unmasked）注意力有强理论保证，但因果注意力近似在理论上存在空白，现有方法（如HyperAttention、BalanceKV）的保证较弱或资源开销大。
- **整体含义**：本文旨在填补因果注意力近似的理论空白，提出一个通用元过程Express，能将任意非因果注意力近似（基于子采样/减薄）高效转换为因果近似，同时保持近似质量保证。结合Thinformer，在理论保证、内存和压缩开销上均优于现有方法，并在实际系统中通过Triton实现显著加速，克服语言模型流程中的四个资源瓶颈。

### 2. 方法论
- **核心思想**：将因果注意力问题转化为一系列动态更新的加权缓存，缓存通过减薄（thinning）算法维护，保证对每个查询的注意力输出近似误差可控。Express元过程利用**子高斯减薄**（sub-Gaussian thinning）理论，通过组合子采样（SUBSAMPLE）、压缩2（COMPRESS 2）和内核减半（Kernel Halving, KH）算法，实现流式减薄。
- **关键技术细节**：
  - **Express算法（Alg.1）**：初始化空缓存，经历三个阶段：**精确阶段**（存储前$n_{\text{out}}$个点）、**减薄阶段**（将后续点分批，每组大小$2^{m}n_{\text{out}}$，通过SUBSAMPLE + COMPRESS 2逐步减至$n_{\text{out}}$）、**减半阶段**（当缓存大小达到$4n_{\text{out}}$时，应用两次内核减半（KH）将其缩减至$n_{\text{out}}$并增大减薄因子$m$）。
  - **COMPRESS 2（Alg.2）**：递归减薄，将输入分成几何级数增长的组，每组独立调用KH。
  - **SUBSAMPLE（Alg.3）**：分层均匀采样，每$2^{m-m}$个点中随机保留一个。
  - **Thinformer Express（Alg.4）**：实例化KH-Express，使用注意力核$\kappa_{\text{att}}((k,v),(k',v')) = \exp(\langle k,k'\rangle/\sqrt{d})(\langle v,v'\rangle + v_{\max}^2)$，为每个新token调用WTD_ATTENTION（Alg.5）在缓存和当前token上计算近似注意力。
